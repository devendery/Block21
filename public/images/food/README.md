# Food Images for Block21 Worms

## Required Images

Place the following PNG images in this directory (`/public/images/food/`):

1. **apple.png** - Red apple (recommended: 16x16 to 32x32px)
2. **berry.png** - Purple/blue berry (recommended: 16x16 to 32x32px)
3. **coin.png** - Gold coin (recommended: 16x16 to 32x32px)
4. **gem.png** - Cyan gem (recommended: 16x16 to 32x32px)
5. **star.png** - Yellow star (recommended: 16x16 to 32x32px)
6. **heart.png** - Pink heart (recommended: 16x16 to 32x32px)
7. **diamond.png** - Blue diamond (recommended: 16x16 to 32x32px)
8. **crystal.png** - Purple crystal (recommended: 16x16 to 32x32px)

## Image Specifications

- **Format**: PNG with transparency
- **Size**: 16x16px to 32x32px (will be scaled in game)
- **Background**: Transparent
- **Style**: Simple, colorful, matching worms.zone aesthetic

## Fallback

If images are not found, the game will automatically use colored circles matching the food type colors:
- Apple: #ff4444 (red)
- Berry: #ff88cc (pink)
- Coin: #ffd700 (gold)
- Gem: #00ffff (cyan)
- Star: #ffff00 (yellow)
- Heart: #ff69b4 (hot pink)
- Diamond: #b9f2ff (light blue)
- Crystal: #9370db (medium purple)

## Quick Image Generation

You can use any image editor or online tool to create simple food icons:
- [Canva](https://www.canva.com) - Free icon creation
- [Flaticon](https://www.flaticon.com) - Free icons
- [Icons8](https://icons8.com) - Free icons
- Simple pixel art tools

## Testing

After adding images, restart the dev server and check the browser console for any loading errors.
